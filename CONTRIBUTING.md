# Contributing to Millete API

Thanks for your interest in contributing.

## How to contribute

1. Fork the repository
2. Create a branch: `git checkout -b feature/name`
3. Make your changes and commit: `git commit -m "feat: description"`
4. Push: `git push origin feature/name`
5. Open a Pull Request

## Commit convention

We use [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation changes
- `refactor:` code changes without new functionality
- `test:` adding or updating tests

## Reporting bugs

Use the Bug Report template in Issues.

## Code of conduct

Be respectful. We do not tolerate harassment.
